# Bot para o Telegram a título de aprendizagem

Nesse projeto construi um código simples para demonstrar como funciona um Bot no Telegram.

# Como rodar o BOT

1) Entre no grupo do telegram criado de teste:
https://t.me/+E86jrgz8AzY1YzAx

2) Baixe o código do Github e aperta F5

3) O bot só intercepta os seguintes comandos

/carteira
/staking
/saldo

Veja mais no código C#.
